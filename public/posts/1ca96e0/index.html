内容简介

分享一些 Windows 中好用且日常会用到的软件

日常办公

解压缩软件

7-Zip

7-Zip

只有几MB，解压缩速度都不低，免费使用，主打简洁。

Bandizip

Windows 版本的 Bandizip

免费使用，功能很齐全；但没有购买许可证的情况下经常会提示更新。

桌面截图软件

PixPin

PixPin 截图

大部分功能都免费，支持长截图和动态截图，功能强大。(更推荐)

Snipaste

Snipaste

老牌截图软件，和PixPin没有很大区别。

桌面翻译软件

重要

相较于使用翻译网站的好处是：阅读文档或者是想翻译图片文字，这类软件都很方便，支持划词和OCR翻译

以下两款各有优势，暂未发现哪个更好。

TTime

TTime | 截图翻译、划词翻译软件

支持的翻译源非常多，用起来方便；可以创建AI应用(如代码解释器)。

Pot

Pot | 跨平台划词翻译和OCR

功能更强大，支持多种插件(包括离线文件识别、语音合成、生词本……)。

文件搜索

Listary

Listary – Free File Search Tool

界面美观，启动很快。(更推荐)

EveryThings

voidtools

完全免费，没啥明显缺点。

AnyTXT Searcher

https://anytxt.net/

这款软件可以搜索文件内的内容，非常强大！(还没用过)

Windows 管理

桌面整理

Coodesker

酷呆桌面 - 桌面整理工具

目前已知的唯一真神，后台占用只有几MB，非常给力。

右键菜单管理

火绒安全

个人产品-火绒安全

这是一个杀毒软件之外，还可以管理右键菜单，十分干净简洁。 如果不想装火绒，这个右键菜单管理好像还有单独版本，这里就不提供了。

ContextMenuManager

https://github.com/BluePointLilac/ContextMenuManager

该软件已经有几年没有更新了，不过用起来还没有问题。

软件卸载

Revo Uninstaller Pro

Revo Uninstaller Pro

有一个 猎人模式 可以锁定一些未知进程的窗口，然后直接卸载。 软件本身付费，但网上有大量绿色版。

BCUninstaller

Bulk Crap Uninstaller

有一个可视化的存储占用。

HiBit Uninstaller

HiBit Uninstaller (hibitsoft.ir)

与RevoUninsaller的能力差不多，但软件完全免费。

Geek Uninstaller

Geek Uninstaller

相对于上述两款卸载软件，Geek少了删除浏览器插件等功能，但好在界面很简洁。

存储空间分析

空间不够用了可以用来分析是什么占用了大量存储

Spaceaniffer

https://spacesniffer.org

可以可视化分析存储空间的占用情况，非常好用。

WizTree

WizTree - The Fastest Disk Space Analyzer

听说比spaceniffer更好用。(推荐)

磁盘管理工具

DiskGenius

DiskGenius官方网站

DiskGenius可以管理Windows磁盘分区，这对于安装新系统时非常有用。另外很有用的一点是，可以查看到隐藏的分区，比如U盘接入电脑后没有在资源管理器显示的情况，有可能是U盘因为没有分区等情况无法显示，并不是U盘损坏。 还包括一些数据恢复工具(没用过)。

垃圾清理

CCleaner

CCleaner Professional

清理能力非常强，可以清理浏览器缓存以及注册表无效的索引…… 虽然官网有免费版本，但是推荐网络上的绿色便携版，不要安装版。

U盘启动盘制作

微PE工具箱

超好用的装机维护工具

如果只考虑制作Windows的启动盘则可以无脑选择微PE，操作很简单。

Ventoy

Ventoy

传说中“最强的U盘启动工具”，事实应该也是这样。这款软件支持同时制作多个启动镜像在一个U盘内。网上有人说不太稳定，但我还没遇到过。

Rufus

Rufus - 轻松创建 USB 启动盘

能制作Linux 和Windows 启动镜像，缺点是不能像Ventoy一样在U盘启动时有多个iso可以选择。

其他

图吧工具箱

图吧工具箱 - 最纯净的硬件工具箱 (tbtool.cn)

电脑小白神器！包含了CPU、主板、内存等各种硬件的检测工具。

视频播放

PotPlayer

PotPlayer高清影音播放器

非常牛的视频播放器，支持很多解码格式，比微软自带的更实用。

视频录制

OBS Studio

Open Broadcaster Software | OBS

免费且非常专业的视频录制软件，就是太高级了，用不习惯。

置顶窗口

DeskPins

https://efotinis.neocities.org/deskpins/

该软件可以将任何窗口实现置顶，让这个窗口始终处于最上层，在屏幕小且要同时查看某些窗口时很有用。
